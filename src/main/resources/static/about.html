Tentang Flame

Platform rekomendasi dan diskusi game untuk para gamer Indonesia

Apa itu Flame?

Flame adalah platform yang membantu kamu menemukan game terbaik berdasarkan genre, platform, dan rating. Data game diambil dan disinkronkan dari basis data game terbesar, lalu disajikan dengan antarmuka yang sederhana dan cepat.

Selain menemukan rekomendasi, Flame juga menjadi tempat diskusi antar pemain melalui forum di setiap halaman detail game.

Tim di Balik Flame

Flame dikembangkan sebagai proyek pembelajaran integrasi backend Spring Boot dengan frontend modern, menggabungkan data eksternal dan fitur komunitas dalam satu aplikasi.
